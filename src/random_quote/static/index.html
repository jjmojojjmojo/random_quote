Random Quote API

Welcome to the Random Quote API. This API provides random quote functionality via HTTP.

API Endpoints

All endpoints only work with GET requests. All responses are JSON

/quote/1
Retrieve a specific quote by its ID. In this example, we are retrieving quote #1.
/quotes
Retrieve all quotes.
/random
Retrieve a random quote
/qotd
Retrieve today's "Quote of the day."
/qotd-history
Retrieve all previous "Quote of the day." entries.
